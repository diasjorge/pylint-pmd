==========
Pylint PMD
==========

About
=====

Generate PMD reports for pylint

Installation
============

.. code-block:: bash

   pip install pylint-pmd

Usage
=====

.. code-block:: bash

   pylint -f pylint_pmd.PMDReporter YOUR_PACKAGE
